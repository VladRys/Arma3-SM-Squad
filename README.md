# Arma3-SM-Squad
Personal project of the virtual international squad SMERSH A3.

A set of automation bots built for managing a game clan.
They centralize and automate key clan-related information to reduce manual coordination.

The bots collect, process, and publish data to Discord and Telegram in real time.

## Features
**Game Schedule Parsing**
  
  - Automatically parses upcoming game events.
  - Displays dates and time ranges.

**Clan Slot Tracking**
  - Shows clan-occupied slots in each mission.
  - Groups data by missions and sides.

**Server Status Monitoring**
  - Checks game server availability.
  - Reports current status (online / offline).

**Platform Support**
  - **Discord Bot**
    - Notifications
    - Commands for schedule and status

  - **Telegram Bot**
    - Quick access to the same data
    - Lightweight alternative to Discord
